This project is a website for presenting stocks information like the tradinview website.
The initial frontend was developed on the v0.dev website and is located inside the ./frontend folder.
This project should contain a backend in python which will:
1. pull socks information from the Alpha Vantage web api.
2. return the stocks information to the frontend that will present the reslts
3. pull general stock market info e.g. S&P 500, Dow Jones, NASDAQ ect. that will be shown on the frontend side
4. calculate linear regression channels for the stocks among other calculations and return the info to the frontend to present on the graphs.
